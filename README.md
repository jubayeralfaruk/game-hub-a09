The purpose of the GameHub project is to create a user-friendly platform where users can explore, discover, and manage their favorite games in one place. This website aims to provide detailed game information, ratings, and categories to help users find games based on their interests. It also includes features like user registration, login authentication, Google login, and personalized game libraries to enhance user experience. GameHub is built to make game discovery simple, engaging, and visually appealing for gamers of all ages.

1. Game Listing Page

5. User Profile & Edit Profile
6. Data Loading & Loader Support
7. Responsive Design
8. Error & Not Found Pages
